## 📦 安装与使用

使用cra-template初始化项目

``` 
npx create-react-app --template cra-template-gtja-cz my-app
```

启动

```cd my-app
yarn start
```

## 🚑 无法安装？

如果遇到网络问题无法安装，可以先下载模板全局安装在本地，在执行上面安装操作  

#### 安装模板  
```
yarn add cra-template-gtja-cz -g
```
